About This Project

This project is a Machine Learning based Spam Message Classification System developed using Python, Flask, Scikit-learn and Natural Language Processing (NLP).

Project Objective

The objective of this project is to classify SMS or Email messages as Spam or Ham using Machine Learning techniques.

Technologies Used

  • Python
  • Flask
  • Pandas
  • NumPy
  • Scikit-learn
  • TF-IDF Vectorizer
  • Multinomial Naive Bayes
  • HTML
  • CSS
  • Bootstrap 5

Machine Learning Workflow

  1. Load Dataset
  2. Preprocess Data
  3. Convert Text into TF-IDF Features
  4. Train Naive Bayes Model
  5. Predict Spam or Ham

Model Accuracy

The trained model achieved approximately 96% accuracy on the SMS Spam Collection dataset.
